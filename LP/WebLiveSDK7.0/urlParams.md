# url param
| url参数      | value | 说明                              |
| ------------ | ----- | --------------------------------- |
| sdkStore     | true  | window挂载store                   |
| storeDebug   | true  | 开启mobx的spy功能，监听所有action |
| extDaily     | true  | 开启日志                          |
| extTestSpeed | true  | 开启测速                          |
| extControls  | true  | 开启player控件                    |
| controls     | true  | 开启video原生控件                 |
| poster       | true  | 开启海报                          |
| autoplay     | true  | 开启自动播放                      |
| sdkDebug     | list  | 开启sdk内部执行逻辑在控制台的打印 |
